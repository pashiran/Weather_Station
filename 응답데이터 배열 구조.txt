🌤 단기예보 API 응답 데이터 배열 구조 제안
📌 목적
초단기실황조회, 초단기예보조회, 단기예보조회 API 데이터를 사람이 보기 쉽게 배열화하고, 전자잉크 디스플레이에 적용할 수 있도록 형식 정리.

1. 초단기 실황조회 (getUltraSrtNcst)
특징: 현재 시점의 관측값
시간 해상도: 1시간 단위 (단일 시점)
배열 구조: 항목별 나열

관측시간: YYYY-MM-DD HH:MM

항목    | 값     | 단위
-----------------------
기온     | 16.5°C | ℃
강수량   | 없음   | mm
습도     | 72%    | %
풍속     | 2.3    | m/s

응답 데이터
+----------+------------------------------+-----------+
| category | 설명                         | 단위      |
+----------+------------------------------+-----------+
| PTY      | 강수형태 (비/눈 여부 등)     | 코드값    |
| REH      | 습도                         | %         |
| RN1      | 1시간 강수량                 | mm        |
| T1H      | 기온                         | ℃         |
| UUU      | 동서 바람 성분               | m/s       |
| VVV      | 남북 바람 성분               | m/s       |
| VEC      | 풍향                         | deg (도)  |
| WSD      | 풍속                         | m/s       |
+----------+------------------------------+-----------+


PTY 코드값
+-----+---------------------+
| 값  | 의미                |
+-----+---------------------+
| 0   | 없음                |
| 1   | 비                  |
| 2   | 비/눈 (진눈깨비)    |
| 3   | 눈                  |
| 4   | 소나기              |
| 5   | 빗방울              |
| 6   | 빗방울/눈날림 혼합 |
| 7   | 눈날림              |
+-----+---------------------+


2. 초단기 예보조회 (getUltraSrtFcst)
특징: 30분 간격, 6시간 예보

시간 해상도: 30분
배열 구조: 시간 X축, 항목 Y축

시간       | 06:30 | 07:00 | 07:30 | ...
--------------------------------------
기온(T1H)   | 15.8  | 16.0  | 16.3
강수확률(POP)| 20%   | 30%   | 40%


+----------+----------------------------+-----------+-------------------------------+
| category | 설명                       | 단위      | 비고                          |
+----------+----------------------------+-----------+-------------------------------+
| LGT      | 낙뢰 가능성                | 코드값    | 0: 없음, 1~3: 있음            |
| PTY      | 강수 형태                  | 코드값    | 위의의 별도 표 참조             |
| PCP      | 1시간 강수량               | mm        | 강수없음, 1mm 미만 등 텍스트 |
| REH      | 습도                       | %         |                               |
| SNO      | 1시간 신적설               | cm        | 적설없음, 1cm 미만 등 텍스트 |
| SKY      | 하늘 상태 (구름 양)        | 코드값    | 1: 맑음, 3: 구름많음, 4: 흐림 |
| T1H      | 기온                       | ℃         |                               |
| TMN      | 일 최저기온                | ℃         | 단기예보에서 제공            |
| TMX      | 일 최고기온                | ℃         | 단기예보에서 제공            |
| UUU      | 동서 바람 성분             | m/s       |                               |
| VVV      | 남북 바람 성분             | m/s       |                               |
| VEC      | 풍향                       | deg       | 0~360도                       |
| WSD      | 풍속                       | m/s       |                               |
+----------+----------------------------+-----------+-------------------------------+



3. 단기예보조회 (getVilageFcst)
특징: 3시간 간격, 최대 4일 예보

시간 해상도: 3시간
배열 구조: 시간 X축, 항목 Y축 (최고/최저기온은 하루 1회)

시간       | 09:00 | 12:00 | 15:00 | ...
───────────────
기온(TMP)   | 18.0  | 21.0  | 20.5
하늘상태(SKY)| 맑음  | 흐림   | 비
